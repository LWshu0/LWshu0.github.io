# 内存分区

从语言角度分为 5 大分区, 从低地址到高地址依次如下:

1. 代码段: 存储编译后生成的代码指令, 只读常量. 它是只读的, 防止代码被修改.
2. 常量区: 常量(例如常量字符串, 常量整数等).
3. 静态区: 存储全局变量, 静态变量. 静态区分为 BSS 和 DATA 两个部分.
    - BSS段: 存储未初始化的全局变量和静态变量, 程序开始时自动初始化为零. 全局变量和静态变量与程序生命周期一致.
    - DATA段: 存储已初始化的全局变量和静态变量, 程序启动时被赋值.
4. 堆区: 动态分配内存区域.  通过 new 和 delete 操作符管理, 内容由程序员手动管理, 生命周期由程序员决定.  
5. 栈区: 存储局部变量和函数调用信息, 例如: 函数运行时而分配的局部变量, 函数参数, 返回值, 返回地址, 指针, const局部变量. 每当一个函数被调用时, 栈区分配内存, 当函数返回时, 内存被释放, 由程序自动管理.
